项目地址：https://github.com/gohugoio/hugo

以下环境使用：树莓派Linux kali 4.19.127-Re4son-v8l+

安装hugo

$ apt-get install hugo

生成博客

$ hugo new site myblog   #指定博客名称为myblog
  • 此时，站点已经生成，在当前目录下会有myblog这个目录，里面存放了博客配置信息。

设置主题

官方主题库：https://themes.gohugo.io/

此次演示：https://themes.gohugo.io/hugo-theme-m10c/

$ cd myblog && git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c

更改配置

$ echo theme = "\"m10c"\" >> config.toml

启动服务

$ hugo server -t m10c --buildDrafts

写一遍文章

$ hugo new post/blog.md
