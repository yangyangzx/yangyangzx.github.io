Go特性

1.既有静态编译语言的安全和性能，又有动态语言开发维护的高效率，Go=C+Python，继承了C的理念包括表达式语法，控制结构，基础数据类型，调用参数传值，指针，保留了C语言一样的编译执行方式及弱化的指针。

2.引入包的概念，Go语言的一个文件都归属一个包，而不能单独存在。

3.垃圾回收，内存自动回收。

4.支持高并发，Goroutine轻量级线程实现大并发，高效使用多核处理，基于CPS并发模型实现。

5.吸收了管道通信机制，形成Go语言特有的Channel，实现不同的Goroutine之间的相互通信。

6.函数返回多个值。

7.增加了切片slice，延时执行defer等。

Go开发注意事项

1.Go程序的源代码是以.go为后缀；

2.Go应用程序的执行入口是main函数；

3.Go语言严格区分大小写；

4.Go的方法是由一条一条语句构成，每个语句后面不需要加;号（Go会自动添加）；

5.Go编译器是逐行编译，每一行就写一条语句，不能把多条语句写在同一行，否则报错；

6.Go语言定义的变量或者import的包必须被使用到，否则编译不通过；

7.{}括号是成对出现的，缺一不可，第一个{写在函数名称同一行末;

8.推荐使用行注释，注释整个方法或语句;

9.正确的缩进和空白;

10.运算符两边加空格，如：num = 2 + 4 * 2;

11.一行的长度不超过80个字符对于超长的字符可以使用以下方法（假设以下为超长字符）。

fmt.Println("hello",			"hello",            "hello")    #最后一个不使用,号

字符型

  • Golang中没有专门的字符类型，如果存储单个字符（如：字母），一般都使用byte来保存。
  • 字符串是遗传固定长度的字符连接起来的字符序列，GO的字符串是由字节组成的。

字符类型的演示

func main() {    var a1 byte = '8'    //在输出时byte类型的数据值为ASCII码中对应的值	var b1 byte = 'c'    //如果希望输出字符本身，需使用格式化输出	fmt.Println(a1, b1)	fmt.Println("a1类型是:", a1, unsafe.Sizeof(a1))	fmt.Println("b1类型是:", b1, unsafe.Sizeof(b1))    fmt.Printf("a1=%c b1=%c", a1, b1)   //此处使用了格式化输出字符本身}
  • 如果字符在Asci表中，比如[0-1,A-Z...]可直接使用byte；
  • 如果字符对应的值大于255，需使用int类型的保存；
  • 如果需要显示字符本身，则需要格式化输出，即fmt.Printf("%c", 变量名称)
  • Golang的字符串使用Unicode(UTF-8)编码格式，常量赋值时可查看UTF-8编码表查看对应的Unicode编码

字符类型的细节

  • 字符常量需使用('')单引号，括起来单个字符，如：var a1 byte = 'a' var c2 int = '中' var b3 byte = '9'
  • Go语言的字符使用UTF-8编码，英文字母是1个字节，汉字是3个字节；
  • Go中，字符的本质是一个整数，直接输出时，是该字符对应的UTF-8编码的码值；
  • 可以直接给一个变量赋值一个数字，然后格式化输出时使用%c，可直接输出该数字对应的Unicode字符。
  • 字符类型可参数运算，相当于一个整数，因为都有对应的Unicode码。

变量（值类型）

  • 变量表示内存中的一个存储区域，这个区域有变量的名称和数据类型；
  • 在Go语言中所有的变量声明或者赋值后必须使用，否则就会报错。
  • 变量的三要素变量=变量名+值+数据类型
  • 如果变量没赋值，变量的默认值为int=0,string=空，bool=false

名称要求

  • 变量名称必须只包含字母,数字,下划线
  • 不能以数字开头
  • 不能使用Go语言内置的关键字

使用注意事项

  • 指定变量类型，声明后不赋值，使用默认值
  • 根据值自行判断变量类型（类型推导）
  • 省略var，:=左侧的变量名不应该是已经声明过的，否则会导致编译错误
  • 变量在作用域内的数据值可以在同一类型范围内不断变化
  • 变量在同一个作用域内不能重名（如出现两个名为a的变量名）
  • 变量使用中，遵守保小不保大的原则，在保证程序正确运行的情况下，尽量使用占用空间小的数据类型。如age这个变量不建议使用int64，因为 int64占用的空间更多-2的63次方~2的63次方-1使用byte类型就足够.,

变量作用域

  • 如果我们{}定义了一个域，那么花括号内的变量，将不能被自己的上级所调用，可以在同级使用，也可以调用上级变量
package mainimport "fmt"func main() {    var (    	name = "杨中校"    	age = 29    	massage string    	changdu int    	shazima bool    )                                         //main函数自身的变量体    fmt.Println(name,age,massage,changdu,shazima)    if true {                   //main内的第一个作用域，使用{}定义内的变量    	num := 30              //此变量只能在作用域内使用        nmae := "邓佳艳"    	fmt.Println(num,name,age)  //但域内可以调用上级main函数的一级变量	}	fmt.Println(num)           //这个调用方法将会报错，因为num变量属于main内的作用域，所以不能被调用}
